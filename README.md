# Spring Boot Personal Project

## 💾프로젝트

스프링부트 환경 ( Gradle ) 에서 마이바티스와 Ajax를 이용한 Rest API 회원 게시판 개인 프로젝트
<br><br><br>

### 🕓기간

- **24.02.10** ~ **24.02.13** <br><br><br>


## 📝기능

### 1️⃣회원가입 및 로그인

* Ajax를 이용한 이메일 중복 자동 체크<br>
* DB 연동으로 로그인 이메일과 비밀번호 체크<br>
* 로그인 시 게시판으로 가는 버튼 생성<br><br>

### 2️⃣게시판

* 게시판 CRUD 기능
